Show that there exists a function that is not time-constructible.

\textbf{Answer:}

Consider the Busy Beaver function $BB(n)$.

We know that $BB(n)$ is not computable. On the other hand, every time-constructible function is computable, since by definition there is a Turing machine that computes $f(n)$ in time $O(f(n))$.

Hence $BB(n)$ cannot be time-constructible.

Therefore, there exists a function that is not time-constructible.

Hence proved.
