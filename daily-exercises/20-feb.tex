A context-free grammar is linear if each production contains at most one non-terminal. Show that the language of well-balanced parentheses cannot be generated by a linear grammar.
\textbf{Answer:}

Let $G = (V, \Sigma, P, S)$ be a linear grammar. WLOG, assume that the production rules are simplified to the following form:
\begin{itemize}
    \item $A \to aB$
    \item $A \to Ba$
    \item $A \to \epsilon$
    \item $A \to a$
\end{itemize}

Let $L_1$ be the language generated by $G$. Let $L_2$ be a regular language and $A$ a DFA for $L_2$.

\textbf{Claim:} $L_1 \cap L_2$ is linear. I.e linear languages are closed under intersection with regular languages.

\textbf{Proof:}

I will construct a linear grammar $G'$ for $L_1 \cap L_2$. Let $G' = (V', \Sigma, P', S')$ be a linear grammar. The non-terminals are defined at triples of the form $(p, A, q)$, where $p, q \in Q$ and $A \in V$. Define the production rules for each production rule from $G$ as follows:
\begin{itemize}
    \item $A \to aB$:\\
    Look at all states $p \in Q$ s.t $\delta(p, a) = p'$ for some $p' \in Q$ (i.e it does not fail here). For each such $p$, add the production rule $(p, A, q) \to a(p', B, q)$ for all $q \in Q$.

    \item $A \to Ba$:\\
    Look at all states $q \in Q$ s.t $\exists q' \in Q$ s.t $\delta(q', a) = q$ (i.e there exists a way to get to $q$ by reading $a$). For each such $q$, add the production rule $(p, A, q) \to (p, B, q')a$ for all $p \in Q$.

    \item $A \to \epsilon$:\\
    Add the production rule $(p, A, p) \to \epsilon$ for all $p \in Q$.

    \item $A \to a$:\\
    Add the production rule $(p, A, q) \to a$ for all $p$ and $q = \delta(p, a)$.
\end{itemize}

The start symbol is $S' = (q_0, S, q_f)$ for all $q_f \in F$, where $q_0$ is the start state of $A$.

This construction is linear, and $L(G') = L_1 \cap L_2$ (can prove, do not prove here for Brevity).

We have now established that linear languages are closed under intersection with regular languages. Suppose that the language of well-balanced parentheses can be generated by a linear grammar $G$. Now, consider the regular language $L_2 = \left\{ a^+b^+a^+b^+ \right\}$. $L_2$ is regular, so $L_1 \cap L_2$ must be linear.
